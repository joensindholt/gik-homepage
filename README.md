# gik-homepage

Homepage for Gentofte Athletics Club

## Running it
 
Clone it
```
git clone ...
```

Compile and watch the scss files. Ie. *node-sass-watcher*
```
node-sass-watcher scss/styles.scss -o styles.css -c 'node-sass <input>'
```

Serve it - ie. with the npm *serve* package
```
serve .
```
